## **如何使用？**
### **点开Releases下载**
### **打开程序**
### **将要识别的图像名字改成target_image**
### **将改名的图像扔进程序运行目录的Config\picture文件夹(第一次启动程序时生成)**

## **常见问题**
### **问：鼠标失控怎么办**
### **答：按下F8（没用就多按几次）**
